# PitchEye
Lightweight, smartphone-based system for recording live soccer matches using multiple volunteer devices. Users join a game session through a QR code and capture short video clips from the sidelines.

### Getting Started
Documentation and setup instructions coming soon as we build out the different components of the system.

### Tech Stack
Backend: Python (Django), C++
Frontend: TypeScript/JavaScript (React)
Video Processing: FFmpeg, OpenCV
Streaming: WebRTC / RTMP
Database: Supabase (PostgreSQL)
Storage: Supabase Storage
Containerization: Docker
